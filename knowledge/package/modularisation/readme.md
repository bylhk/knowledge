# Modularisation
Modularisation can help us to avoid duplicate codes and long scripts.

## Example
```
class BasicModule:
    def many_functions():

class FirstTypeOfModule(BasicModule):
   def __init__(x):
       super().__init__(x)
       # custom_script

class SecoundTypeOfModule(BasicModule):
   def __init__(x):
       super().__init__(x)
       # custom_script
```
